This is the official repository for my personal portfolio website, [shahaduddin.com](https://shahaduddin.com). The site is designed to showcase my journey as a Mathematics student and a programmer, featuring my projects, technical skills, academic background, and thoughts on code and creativity.

## ✨ Features

*   **Clean, Modern Design:** A minimalist and professional interface with smooth animations powered by Framer Motion.
*   **Project Showcase:** A dedicated section highlighting my key projects with descriptions.
*   **Skills Arsenal:** A categorized overview of my technical skills, from frontend and backend to scientific computing tools.
*   **Academic & Professional Context:** Information about my current studies in Mathematics at Shahjalal University of Science and Technology and my availability for opportunities.
*   **Blog/Thoughts Section:** A space to share articles on numerical methods, programming, and my learning experiences.
*   **Direct Contact:** Clear calls-to-action for collaboration, including a resume download link and a "Send Me a Message" prompt.

## 🛠️ Built With

*   **Frontend Framework:** [React](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for utility-first styling
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) for smooth page and UI animations
*   **Deployment:** [Vercel](https://vercel.com/) for seamless hosting and continuous deployment

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have **Node.js** (which includes npm) installed on your machine.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/shahaduddin/shahaduddin.github.io.git
    ```
2.  **Navigate to the project directory**
    ```bash
    mv shahaduddin.github.io your-name
    cd your-name
    ```
3.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```
4.  **Start the development server**
    ```bash
    npm start
    # or
    yarn start
    ```
5.  Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will reload when you make changes.
